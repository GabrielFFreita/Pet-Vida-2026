# Fluxo do Sistema

## Fluxos publicos

- `index.php`: vitrine principal e destaques.
- `adocao.php`: listagem publica de animais.
- `quiz.php`: quiz de combinacao de perfil com pets.
- `login.php` e `cadastro.php`: autenticacao.
- `perfil.php`: manutencao da conta do usuario logado.

## Fluxos administrativos

- `admin/dashboard.php`: visao geral do painel.
- `admin/animais.php`: listagem administrativa de animais.
- `admin/animal-editar.php` e `admin/animal-excluir.php`: manutencao de animais.
- `admin/abrigos.php` e `admin/abrigo-perfil.php`: gestao de abrigos.
- `admin/usuarios.php`, `admin/usuario-editar.php` e `admin/usuario-excluir.php`: gestao de usuarios.

## API

- `api/index.php`: autenticacao, sessao, favoritos, perfil e listagens.
- `api/solicitar-adocao.php`: solicitacao de adocao.
- `api/upload.php`: fluxo legado de upload.
